# ON-BOARD-CAMERA-SYSTEM-DEVELOPMENT-FOR-DEBRIS-DETECTION-AND-DECISION-MAKING

ON BOARD CAMERA SYSTEM DEVELOPMENT FOR DEBRIS DETECTION AND DECISION MAKING
Overview
An AI-powered onboard camera system designed to autonomously detect hazardous space debris and execute collision avoidance maneuvers for satellites in Low Earth Orbit (LEO), without human intervention.
The Problem
Space debris in LEO travels at extremely high speeds, making ground-based monitoring unreliable due to sunlight dependency and detection limitations for small objects. A faster, autonomous onboard solution is needed.
Our Solution
A subsystem installed inside the satellite that:

Uses an onboard camera to continuously monitor surrounding space
Detects debris based on location, size, shape, and movement using Computer Vision
Makes real-time autonomous decisions to avoid collision through one of three strategies:

Rotating the satellite to absorb impact on a shielded side (Whipple Shield)
Adjusting attitude while maintaining the same orbit
Modifying orbital velocity to pass earlier or later at a specific point



Tech Stack

Language: Python
Domain: Computer Vision, AI Decision Making, Orbital Mechanics

Results

Successfully detected moving objects using onboard camera in simulated environment
Autonomous decision-making system with 300+ lines of logic
Real-time attitude adjustment based on debris trajectory
